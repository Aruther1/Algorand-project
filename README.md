# Algorand-project

A blockchain development project built on the Algorand network as part of coursework.

## Project Overview
This project explores building decentralized applications (dApps) on the Algorand blockchain using smart contracts and the Algorand SDK.

## Technologies Used
- Algorand SDK (Python)
- PyTeal / AlgoKit
- Python 3

## Goals
- Understand Algorand's consensus mechanism
- Write and deploy basic smart contracts
- Interact with the Algorand TestNet
